Responsividade no Flexbox

Responsividade é a adaptação do site para diferentes telas, como celular, tablet e desktop.

Flex-flow (quebra de linha)

  • nowrap: não quebra linha; os itens ficam espremidos.
  • wrap: quebra linha quando necessário.
  • wrap-reverse: quebra linha para cima.

Flex Items

  • align-self: alinha cada item no eixo cruzado (stretch, flex-start, flex-end, center).
  • order: define a ordem dos itens usando números.
  • flex: controla quanto espaço o item ocupa no eixo principal; valor maior ocupa mais espaço.

A prática é essencial para dominar Flexbox. No fim do módulo, projetos reais ajudam a aplicar esse conteúdo.
