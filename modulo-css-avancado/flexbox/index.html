O que é Flexbox?

Flexbox (Flexible Box Module) é uma técnica de layout no CSS para criar layouts flexíveis, dinâmicos e responsivos. Para ativá-lo, define-se display: flex; no elemento pai (Flex Container) — os filhos (Flex Items) se posicionam lado a lado por padrão.

Propriedades principais

  • flex-direction: define a direção — row (padrão, itens lado a lado; main horizontal, cross vertical) ou column (um abaixo do outro; main vertical, cross horizontal).
  • justify-content: alinha no main axis — flex-start, flex-end, center, space-between, space-around.
  • align-items: alinha no cross axis — stretch, flex-start, flex-end, center.
  • align-content: alinha múltiplas linhas — flex-start, flex-end, center, space-between, space-around.

Eixos

  • Main Axis: direção principal — horizontal em row, vertical em column.
  • Cross Axis: direção perpendicular ao main axis.
