Seção 1 — HTML Básico

HTML (HyperText Markup Language) é a linguagem de marcação padrão para criar páginas na web. Com HTML você estrutura o conteúdo.

As tags mais comuns são: <h1> a <h6> para títulos, <p> para parágrafos, <a> para links, <img> para imagens.

O HTML5 trouxe novas tags semânticas como <header>, <main>, <footer>, <section>, <article> e <nav>.

Role para baixo para ver o título desta seção "grudar" no topo e depois ser substituído pelo título da próxima seção.

Seção 2 — CSS Fundamentos

CSS (Cascading Style Sheets) é responsável pela apresentação visual das páginas HTML. Com ele definimos cores, fontes, espaçamentos e layouts.

As três formas de aplicar CSS são: inline (atributo style), interno (tag <style> no <head>) e externo (arquivo .css).

O modelo de caixa (box model) é fundamental: cada elemento tem content, padding, border e margin.

Continue rolando para ver o comportamento sticky se repetir nesta seção.

Seção 3 — Position no CSS

static — padrão, o elemento segue o fluxo normal.

relative — deslocado em relação à posição original; espaço original é preservado.

absolute — removido do fluxo; posicionado em relação ao ancestral com position não-static.

fixed — removido do fluxo; posicionado em relação à viewport; não se move com o scroll.

sticky — híbrido: relative até atingir o limite definido, então age como fixed dentro do seu container. É o que você está vendo agora!
